프로젝트 섹션 목업 비교

랜딩 페이지의 Projects 섹션 4가지 디자인을 비교해보세요.

A — Asymmetric Editorial
좌우 교차 비대칭 그리드
5:7 비율로 텍스트와 이미지가 좌우 교차. Stripe / Linear 사례 페이지 톤. 텍스트가 충분히 드러나는 안정적 패턴.
B — Magazine Spread
잡지 스프레드 + 거대 번호 오버레이
이미지 ⅔ + 텍스트 ⅓. 큰 번호를 이미지 위에 mix-blend-mode로 오버레이. Monocle 매거진 스타일.
C — Timeline + Preview
수직 타임라인 + 인터랙티브 프리뷰
좌측에 4개 프로젝트 타임라인, 우측에 선택된 프로젝트 큰 이미지 + 상세. 호버하면 프리뷰 변경. Vercel 사례 스타일.
D — Dense 2x2 Grid
조밀 2×2 그리드
2x2 그리드에 4개 프로젝트 동시 노출. 호버 시 이미지 zoom + 번호 슬라이드. Notion 갤러리 스타일.
