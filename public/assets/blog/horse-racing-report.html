Pilot Study

실시간 배당 데이터와 정적 경주 데이터의
앙상블 기반 경마 순위 예측: 파일럿 스터디

우혜빈 (Hyebin Woo) 2026년 3월 29일

Abstract

본 연구는 한국마사회(KRA) 서울경마장 데이터를 활용하여, 정적 경주 데이터(출마표)와 동적 시장 데이터(실시간 배당률)를 결합한 경마 순위 예측 모델의 가능성을 실증적으로 탐색한다. 2026년 3월 29일 서울경마장 4개 경주(5R~8R)에 대해 실전 검증을 수행한 결과, 정적 데이터 단독 모델 대비 배당 변화율을 포함한 앙상블 모델이 3착 이내 적중률에서 우수한 성능을 보였다. 특히 마감 직전 배당 급등 현상("Late Money")이 강력한 예측 신호임을 확인하였으며, 거리 적성 피처가 정적 피처 중 가장 높은 중요도를 보였다. 복연승 기반 리스크 관리 전략의 효과도 실증하였다.

1. 서론

경마 예측은 스포츠 분석, 금융 시장 모델링, 정보 경제학이 교차하는 흥미로운 연구 분야이다. 경마 시장은 패리뮤추얼(parimutuel) 구조를 갖추고 있어 시장 참여자들의 집단 지성이 배당률에 반영되며, 이는 효율적 시장 가설의 실증적 검증 무대로도 활용되어 왔다.

기존 경마 예측 연구의 대부분은 정적 데이터(출마표, 과거 성적, 마체중 등)에 의존한다. 그러나 경주 마감 직전의 배당률 변화에는 정적 데이터에 포착되지 않는 실시간 정보—패독(paddock) 상태, 내부 정보, 대규모 베터의 판단—가 반영된다.

본 연구의 목적은 정적 경주 데이터와 동적 시장 데이터(실시간 배당률)를 결합한 앙상블 모델이 단일 데이터 소스 기반 모델보다 우수한 예측력을 보이는지 실증적으로 검증하는 것이다.

연구 질문

"실시간 배당 변화율이 경마 순위 예측력을 유의미하게 향상시키는가?"

2. 데이터

2.1 정적 데이터 (KRA 출마표 PDF)

한국마사회(KRA) 공식 사이트에서 제공하는 출마표 PDF를 9주치(2026.02.28~03.29) 수집하였다. PDF URL 패턴을 활용하여 체계적으로 다운로드하였으며, 다음 피처를 추출하였다:

  • 레이팅(R) — KRA 공식 레이팅
  • 마체중 변화 — 전회 대비 체중 변동
  • 기수 통산승/승률 — 기수의 커리어 성적
  • 조교사 승률 — 조교사의 최근 성적
  • 해당거리 성적 — 승/준/복 횟수
  • 부담중량, 연령, 성별
  • 부상이력, 종반200m 기록

2.2 동적 데이터 (실시간 배당)

마감 30분 전부터 5분 전까지의 배당 시계열을 수집하였다. 핵심 파생 피처로 배당 변화율(Δodds)을 산출하였다:

Δodds = (oddst-5min − oddst-30min) / oddst-30min

배당 변화율이 음수(−)이면 마감 직전 해당 마필에 자금이 유입된 것이며, 이는 잠재적으로 유력한 예측 신호가 된다.

2.3 라벨

실제 경주 결과(1~3착 순위)를 라벨로 사용하였다.

3.2 모델 비교

  • Baseline — 시장 단승 배당 순위를 그대로 예측으로 사용
  • Model A (정적) — 정적 피처 기반 가중 합산 스코어링
  • Model B (동적) — 동적 피처(배당 변화율) 기반 예측
  • Model C (앙상블) — 정적 + 동적 피처의 가중 결합

3.3 리스크 관리

  • 복연승 보험 전략 — 모든 경주에 3착 이내 보험 베팅 포함
  • 투자 분산 규칙 — 1경주 최대 25%로 제한하여 리스크 분산

4.3 배당 변화율의 예측력

6R 사례 분석 — Late Money 현상

③ 화산폭팔

마감 30분전 단승 배당 8.9 → 5분전 2.8

Δodds = −68.5%

결과: 1착

해석: 정보 우위를 가진 베터들의 마감 직전 대규모 베팅("Late Money")이 강력한 예측 신호로 작용. 정적 모델이 이 마필을 예측하지 못한 반면, 배당 변화율이 이를 포착할 수 있었다.

5. 논의

5.1 주요 발견

  1. 동적 데이터(배당 변화율)가 가장 강력한 단일 피처 — Late Money 현상이 정보 비대칭을 반영하며, 이를 활용한 모델이 우수한 성능을 보였다.
  2. 정적 피처 중 거리 적성이 1위 — 해당 거리에서의 과거 성적이 레이팅이나 기수 성적보다 예측력이 높았다.
  3. 레이팅/복승률 단독은 신뢰도 낮음 — KRA 레이팅 단독 사용은 예측력이 제한적이었다.
  4. 앙상블이 3착 이내 적중률에서 최우수 — 정적+동적 결합이 단일 소스 대비 안정적으로 우수한 결과를 보였다.

5.2 한계

  • 샘플 크기 — 4경주만으로는 통계적 유의성을 확보하기 어렵다.
  • 수동 데이터 수집 — 데이터 수집 자동화가 완료되지 않아 확장성에 제약이 있다.
  • 배당 변화율 수집 시점 — 수집 시점이 불규칙하여 피처의 일관성에 영향을 줄 수 있다.
  • 단일 경마장 — 서울경마장, 단일 날짜의 결과이므로 일반화에 한계가 있다.

5.3 향후 연구

  • 9주치 백테스트를 통한 통계적 유의성 확보
  • KRA API 연동을 통한 데이터 수집 자동화
  • ML 모델(XGBoost, LightGBM) 적용 및 정량적 피처 중요도 분석
  • 다중 경마장(부산, 제주) 확장
  • 켈리 기준(Kelly Criterion) 기반 최적 배팅 전략 연구

6. 결론

본 파일럿 스터디를 통해 정적+동적 데이터 앙상블 모델의 가능성을 확인하였다. 특히 마감 직전 배당 변화율("Late Money")이 가장 강력한 예측 신호임을 실증하였으며, 복연승 기반 리스크 관리 전략이 ROI 개선에 효과적임을 보였다.

4경주라는 제한된 샘플에서 Model C(앙상블)가 3착 이내 적중률 100%를 달성한 것은 고무적이나, 향후 대규모 데이터 기반의 정량적 검증이 필요하다. 자동화된 데이터 파이프라인과 ML 모델의 결합을 통해 보다 견고한 예측 시스템 구축이 가능할 것으로 기대한다.
